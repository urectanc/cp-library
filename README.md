# urectanc's library

My competitive programming library

## Usage

Add this library to your project.

```shell
cargo add urectanc --path /path/to/cp-library
cargo add urectanc --git https://github.com/urectanc/cp-library
```

From this repository root, expand or bundle specified crates.

```shell
cargo xtask expand modint union_find
cargo xtask bundle < /path/to/main.rs
```
